# Erledi.ch – Website

## Deployment auf Vercel

### Schritt 1: GitHub
1. Gehe auf github.com und erstelle ein kostenloses Konto
2. Erstelle ein neues Repository namens "erledi"
3. Lade alle Dateien aus diesem Ordner hoch (drag & drop)

### Schritt 2: Vercel
1. Gehe auf vercel.com (bist du schon eingeloggt)
2. Klicke "Continue with GitHub"
3. Wähle dein "erledi" Repository
4. Klicke "Deploy" – fertig!

### Schritt 3: Domain verbinden
1. In Vercel: Settings → Domains
2. Gib "erledi.ch" ein
3. Folge den Anweisungen um DNS bei Infomaniak zu setzen

## Dateien
- index.html – Hauptwebsite
- logo.svg – Erledi Logo
- vercel.json – Vercel Konfiguration
